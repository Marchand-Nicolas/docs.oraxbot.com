# Vote for Orax Plus

import { Steps, Callout } from 'nextra/components'

You can get Orax Plus for free by voting for the bot on Top.gg. Each vote grants approximately one week of Orax Plus.

## How to Vote

<Steps>

### Go to the Dashboard

Open the [Orax Dashboard](https://oraxbot.com/dashboard) and select the server you want to activate Orax Plus for.

### Click "Vote for Orax Plus"

In the Orax Plus panel, click the **"Vote for Orax Plus"** button. This creates a vote intent linking your vote to the selected server.

### Vote on Top.gg

You'll be redirected to the [Top.gg vote page](https://top.gg/bot/812298057470967858/vote). Log in with Discord if needed and cast your vote.

### Orax Plus Activates Automatically

Once Top.gg sends the vote webhook, Orax Plus is activated for your server automatically. The dashboard will poll for the result and update the status.

</Steps>

<Callout type="info">
  If you've voted recently, clicking "Vote for Orax Plus" may activate Orax Plus immediately without redirecting you to Top.gg again.
</Callout>

## How It Works

1. You click "Vote for Orax Plus" in the dashboard for a specific server
2. Orax creates a **vote intent** linking your user ID to the selected guild
3. You're redirected to Top.gg to cast your vote
4. Top.gg sends a webhook to Orax's server with your vote details
5. The webhook matches your vote to the vote intent
6. Orax Plus is activated for the selected server for approximately **one week**

## Vote Cooldown

- You can vote on Top.gg **once every 12 hours**
- Each vote grants ~1 week of Orax Plus
- If you vote regularly, you can maintain Orax Plus indefinitely

<Callout type="tip">
  Always start the voting process from the Orax Dashboard for a specific server. This ensures your vote is automatically linked to the correct server.
</Callout>

## Troubleshooting

| Issue | Solution |
|---|---|
| Orax Plus didn't activate | Wait a few minutes — the webhook may take time. Make sure you completed the vote on Top.gg |
| Wrong server got Orax Plus | Vote intents are per-server. Start a new vote from the correct server's dashboard |
| Orax Plus expired too early | Votes last ~1 week. Vote again to reactivate |
| Vote page didn't open | Allow pop-ups for oraxbot.com, or use the [direct vote link](https://top.gg/bot/812298057470967858/vote) and then click "Vote for Orax Plus" in the dashboard |
