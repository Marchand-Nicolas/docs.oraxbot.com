# Vote for Orax Plus

import { Steps, Callout, Tabs } from 'nextra/components'

You can get Orax Plus for free by voting for the bot on your platform's listing site. Each vote grants approximately **one week** of Orax Plus, and you can vote again to extend it.

The vote provider depends on which platform your server lives on:

- **Discord servers** vote on [Top.gg](https://top.gg/bot/812298057470967858/vote) — the vote is verified automatically via a webhook.
- **Fluxer servers** vote on [Fluxerlist](https://fluxerlist.com/bots/orax) — since Fluxerlist has no webhook, Orax Plus is granted on a trust basis shortly after you open the vote page.

## How to Vote on Top.gg

<Steps>

### Go to the Dashboard

Open the [Orax Dashboard](https://oraxbot.com/dashboard/discord) and select the Discord server you want to activate Orax Plus for.

### Click "Vote on Top.gg"

In the Orax Plus panel, click the **"Vote on Top.gg"** button. This creates a vote intent linking your vote to the selected server.

### Vote on Top.gg

You'll be redirected to the [Top.gg vote page](https://top.gg/bot/812298057470967858/vote). Log in with Discord if needed and cast your vote.

### Orax Plus Activates Automatically

Once Top.gg sends the vote webhook, Orax Plus is activated for your server automatically. The dashboard will poll for the result and update the status.

</Steps>

<Callout type="info">
  If you've voted recently, clicking "Vote on Top.gg" may activate Orax Plus immediately without redirecting you to Top.gg again.
</Callout>

### How It Works

1. You click "Vote on Top.gg" in the dashboard for a specific server
2. Orax creates a **vote intent** linking your user ID to the selected guild
3. You're redirected to Top.gg to cast your vote
4. Top.gg sends a webhook to Orax's server with your vote details
5. The webhook matches your vote to the vote intent
6. Orax Plus is activated for the selected server for approximately **one week**

### Vote Cooldown

- You can vote on Top.gg **once every 12 hours**
- Each vote grants ~1 week of Orax Plus
- If you vote regularly, you can maintain Orax Plus indefinitely

## How to Vote on Fluxerlist

<Steps>

### Go to the Dashboard

Open the [Orax Dashboard](https://oraxbot.com/dashboard/fluxer) and select the Fluxer server you want to activate Orax Plus for.

### Click "Vote on Fluxerlist"

In the Orax Plus panel, click the **"Vote on Fluxerlist"** button. The Fluxerlist vote page opens in a new tab.

### Cast your vote

Log in on [fluxerlist.com](https://fluxerlist.com/bots/orax) if needed and cast your vote for Orax.

### Orax Plus activates after a short wait

Back on the dashboard, a **"Retrieving your Vote on Fluxerlist vote…"** overlay appears for about 20 seconds while you complete the vote. Once the wait ends, Orax Plus is granted to your server for approximately **one week**.

</Steps>

<Callout type="warning">
  Fluxerlist does not send a vote webhook, so Orax cannot verify that a vote
  was actually cast. Orax Plus is granted on a trust basis after the wait —
  please vote honestly so the listing stays useful for everyone.
</Callout>

### How It Works

1. You click "Vote on Fluxerlist" in the dashboard for a specific server
2. The [Fluxerlist vote page](https://fluxerlist.com/bots/orax) opens in a new tab
3. The dashboard shows a "Retrieving vote…" overlay for ~20 seconds
4. Once the wait ends, Orax requests Orax Plus for the selected server
5. Orax Plus is activated for approximately **one week**

### Vote Cooldown

- You can vote on Fluxerlist **once every 12 hours**
- Each vote grants ~1 week of Orax Plus
- If you vote regularly, you can maintain Orax Plus indefinitely

## Troubleshooting

| Issue | Solution |
|---|---|
| Orax Plus didn't activate | Wait a few moments — webhook (Top.gg) or the 20-second retrieval (Fluxerlist) may still be processing. Make sure you completed the vote on the listing site |
| Wrong server got Orax Plus | Votes are applied per-server. Start a new vote from the correct server's dashboard |
| Orax Plus expired too early | Votes last ~1 week. Vote again to reactivate |
| Vote page didn't open | Allow pop-ups for oraxbot.com, or use the direct vote link below and then click the vote button in the dashboard |
