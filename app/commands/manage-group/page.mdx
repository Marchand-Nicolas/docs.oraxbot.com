# /manage-group

import { Callout } from 'nextra/components'

Opens an interactive button-based settings panel for managing the interserv group linked to the current channel.

## Usage

```
/manage-group
```

## Permissions

- **Manage Guild** — The user must have Manage Server permission (or be a group moderator)

## What It Does

Displays an interactive panel with buttons to toggle and configure various group settings:

| Setting | Description |
|---|---|
| **Custom Usernames** | Toggle and configure custom username patterns |
| **Reply Style** | Change how replies are displayed (embed, quote, etc.) |
| **@everyone** | Toggle whether @everyone and @here pings are allowed |
| **Role Mention Sync** | Toggle syncing role mentions across servers |
| **Delete Sync** | Toggle whether message deletions are synced |
| **User Warning** | Toggle the first-time user warning message |
| **Word Blacklist** | Configure blocked words |
| **Message Filters** | Configure advanced filter rules |
| **Moderators** | Manage the list of group moderators |
| **Log Messages** | Configure a log channel for interserv messages |
| **Custom Rules** | Set custom rules text for the `/rules` command |
| **Announcements** | Toggle whether the group receives announcements |

<Callout type="tip">
  All of these settings can also be configured from the [Orax Dashboard](https://oraxbot.com/dashboard) for a more visual experience.
</Callout>

## How to Use

1. Run `/manage-group` in a linked channel
2. The bot responds with a panel of buttons
3. Click a button to toggle or configure that setting
4. Some settings may require additional input via a modal or follow-up prompt

## Notes

- Settings changes apply to **all** channels in the group across all servers
- Per-channel overrides can be set from the dashboard
- See the [Configuration](/configuration) section for detailed explanations of each setting
