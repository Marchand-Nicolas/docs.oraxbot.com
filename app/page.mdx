## About Orax

Orax is a Discord bot that creates **interserver (interserv) chat bridges**. When a user sends a message in a linked channel on Server A, Orax instantly relays it to all other channels linked to the same interserv group on Servers B, C, D, and so on.

Everything works as in a normal channel — you can send, modify, and delete messages, and all of this will be replicated on other Discord servers. Reactions, threads, stickers, and polls are all synchronized too.
