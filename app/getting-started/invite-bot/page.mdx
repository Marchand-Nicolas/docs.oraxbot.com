# Invite the Bot

import { Callout } from 'nextra/components'

To get started with Orax, you need to invite the bot to your Discord server.

## Invite Link

Click the link below to add Orax to your server:

**[Invite Orax to Your Server](https://discord.com/api/oauth2/authorize?client_id=812298057470967858&permissions=533113138240&scope=bot)**

## Required Permissions

When inviting the bot, you'll be asked to grant the following permissions:

| Permission | Why It's Needed |
|---|---|
| **Manage Webhooks** | Orax creates webhooks in each linked channel to relay messages |
| **Send Messages** | To send synced messages from other servers |
| **Embed Links** | To display formatted messages, embeds, and media |
| **Attach Files** | To relay attachments, stickers, and media files |
| **Read Message History** | To sync message edits and deletions |
| **Add Reactions** | To synchronize reactions across servers |
| **Use Slash Commands** | For Orax's slash commands |
| **Manage Messages** | For moderation features and message management |
| **Create Public Threads** | To synchronize thread creation |
| **Send Messages in Threads** | To relay messages within synced threads |

<Callout type="warning">
  The bot must have all the listed permissions to function correctly. If any permission is missing, certain features may not work as expected.
</Callout>

## After Inviting

Once the bot is in your server:

1. **Verify the bot is online** — check that Orax appears in your server's member list
2. **Choose a channel** — decide which channel you want to link to an interserv group
3. **Create or join a group** — head to the next page to learn how to create your first interserv group
